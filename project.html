Pioneering Analysis: Decoding YouTube Reactions to the Barbie Movie with Cutting-Edge Tech

Date: March 21, 2024

Technologies: HTML5, CSS, PowerPoint, Shotcut/iMovie, Affinity Photo, Excel, VBA, Python

Project Description:

Developed a cutting-edge, responsive movie-themed website that features reviews, custom videos, animations, and original content. Utilizing HTML5, CSS, and Affinity Photo, the project showcased a seamless blend of technical prowess and creative web development skills. For the Barbie movie trailer, an extensive analysis of YouTube comments was conducted using Python and VBA to explore audience reactions and sentiment trends. Leveraging Python's advanced natural language processing techniques, the project processed and performed sentiment analysis on the comment data. Additionally, scripts written in VBA fetched comment data from the YouTube API, pre-processed, and visualized it effectively. The analysis focused on the impact of commenter nationality on viewpoints, presenting differences in viewer reactions to the trailer across various countries. This provided critical data support and insights for targeted marketing and audience engagement strategies.

Achievements:

  • Delivered profound insights into audience reactions to the Barbie movie trailer, offering valuable information for marketing strategies and audience targeting.
  • Established a foundation for further research into the behaviors and preferences of different audience segments.
  • Demonstrated the ability to integrate technical and creative skills to develop a dynamic web platform and perform complex data analysis.

Wisconsin Breast Cancer Database Analysis

Date: March 20, 2024

Technologies: Python, VBA

Project Description:

This project focused on the comprehensive analysis of the Wisconsin Breast Cancer Database from the University of Wisconsin Hospital, a dataset provided by Dr. William H. Wolberg. Leveraging Python and VBA, the project delved into clinical data on breast cancer samples to uncover critical insights.

Objectives and Methodology:

Utilizing advanced machine learning techniques, the project aimed to identify significant associations between various clinical features and breast cancer. The analysis facilitated the construction of robust classification models, aiding in precise medical diagnoses. The methodology encompassed a thorough literature review, meticulous data preprocessing, and sophisticated visualization and analysis.

Achievements:

  • In-Depth Analysis: Delivered a detailed exploration of breast cancer diagnosis, offering valuable data support for ongoing medical research.
  • Literature Integration: Integrated and validated findings from existing studies, ensuring the results were both reliable and innovative.
  • Technological Proficiency: Demonstrated adeptness in Python and VBA for data analysis, preprocessing, and visualization.
