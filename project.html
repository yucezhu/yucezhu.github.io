Wisconsin Breast Cancer Database Analysis

Date: March 20, 2024

Technologies: Python, VBA

Project Description:

This project focused on the comprehensive analysis of the Wisconsin Breast Cancer Database from the University of Wisconsin Hospital, a dataset provided by Dr. William H. Wolberg. Leveraging Python and VBA, the project delved into clinical data on breast cancer samples to uncover critical insights.

Objectives and Methodology:

Utilizing advanced machine learning techniques, the project aimed to identify significant associations between various clinical features and breast cancer. The analysis facilitated the construction of robust classification models, aiding in precise medical diagnoses. The methodology encompassed a thorough literature review, meticulous data preprocessing, and sophisticated visualization and analysis.

Achievements:

  • In-Depth Analysis: Delivered a detailed exploration of breast cancer diagnosis, offering valuable data support for ongoing medical research.
  • Literature Integration: Integrated and validated findings from existing studies, ensuring the results were both reliable and innovative.
  • Technological Proficiency: Demonstrated adeptness in Python and VBA for data analysis, preprocessing, and visualization.
